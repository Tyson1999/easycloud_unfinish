1. 私聊管理员获得一个Onedrive账号，自己购买的也可使用
  2. 打开一个浏览器无痕窗口 前往 Onedrive登陆页 登录账号
  3. 回到本系统，点击左侧菜单栏中的"存储策略管理"，输入以下信息
    表单项 值
    策略名 唯一命名，用于识别不同策略，建议 OD-example.com（更换为你的Onedrive邮箱）
    类别 Onedrive
    账号 留空
    Root folder /
    是否激活 是
  4. 点击立即创建
  5. 正常情况下，能在列表中看到你新增的这个策略。找到对应策略，点击下图红色按钮
  6. 点击后会自动跳转到Onedrive授权页，同意授权即可完成对接。
