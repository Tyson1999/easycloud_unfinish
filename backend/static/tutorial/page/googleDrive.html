1. 私聊管理员获得一个Google drive 团队账号，也可自己购买全局管理
  2. 浏览器打开一个无痕浏览（保证当前会话没有登录过其他Google账号！）
  3. 前往Google drive，创建一个根目录（用于存放上传的文件）。 如下图，创建一个"共享盘"，可随意命名。请注意一定要在共享云端硬盘中创建 创建完成后，会自动进入刚刚创建的"共享盘"，如果浏览器没有自动跳转，可双击进入。
  4. 在共享盘中，右键新建一个文件夹，命名为upload
  5. 创建并进入upload文件夹，复制地址栏参数folders/ 后面的那一串代码，注意不要把"/"也复制进去了 比如我这边需要复制的就是17phBQbZmNCjyYeAbYh284A0_nyt57ayB，大家根据自己地址栏的值修改即可。
    记下这一串代码，即为下文要用到的Root folder
  6. 回到本系统，点击左侧菜单栏中的"存储策略管理"，输入以下信息
    表单项 值
    策略名 唯一命名，用于识别不同策略，建议 GD-example.com（更换为你的登录邮箱）
    类别 googleDrive
    账号 留空
    Root folder 上一步记下的值
    是否激活 是
  7. 正常情况下，能在列表中看到你新增的这个策略。找到对应策略，点击下图红色按钮
  8. 点击后会自动跳转到Google授权页，同意授权即可完成对接。
