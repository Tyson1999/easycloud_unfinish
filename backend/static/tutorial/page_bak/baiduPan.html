1. 前往 百度开发者平台 申请接入权限，一般会直接通过无需审核
  2. 登录后 点击此处 右上角创建应用
  3. 在新出现的表单中填写
    表单项 值
    应用类别 软件
    应用名 easycloud_UPLOADER
    应用描述 留空即可
  4. 点击下方创建后，会自动跳转到详情页
  5. 记下如下两个重要参数（AppKey & SecretKey）
  6. 前往：
    https://openapi.baidu.com/oauth/2.0/authorize?response_type=code&client_id=你的AppKey&redirect_uri=oob&scope=basic,netdisk&display=tv&qrcode=1&force_login=0
    获取Oauth2.0授权码
    请复制上述网址内的参数后，替换参数并黏贴到浏览器新窗打开！
  7. 前往：
    https://openapi.baidu.com/oauth/2.0/token?grant_type=authorization_code&code=第6步获得的授权码&client_id=你的AppKey&client_secret=你的SecretKey&redirect_uri=oob
    获取Refresh token。
    请复制上述网址内的参数后，替换参数并黏贴到浏览器新窗打开！
  8. 以上步骤正确的话，会出现如图结果 记下refresh_token，注意不是access_token！
  9. 回到本系统，点击左侧菜单栏中的"存储策略管理"，输入以下信息
    表单项 值
    策略名 唯一命名，用于识别不同策略，建议 baidu-example（更换为你的百度账号）
    类别 baiduPan
    Bucket name 之前记下的->AppKey
    Access key 之前记下的->Refresh token
    Secret key 之前记下的->SecretKey
    是否激活 是
    失效时间 默认

    最后点击立即创建
